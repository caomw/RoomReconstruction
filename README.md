RoomReconstruction
==================

Javier Moreno Bachelor Thesis' source code. Advisers: Javier V. Gómez and David Álvarez

DEPENDENCIES: PCL version 1.7 or higher (1.X), with 2.0 it won't work.

compile with the standar cmake procedure:

$ cd build
$ cmake ..
$ make
